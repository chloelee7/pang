# 게임 룰

## 기본 목표

스테이지 내 모든 풍선(버블)을 제한 시간 안에 하푼으로 터뜨린다.

---

## 플레이어

### 조작
| 키 | 동작 |
|----|------|
| ← / → | 좌우 이동 |
| Space / Z | 하푼 발사 (위 방향 고정) |

### 생명 시스템
- 초기 잔기: **3개**
- 잔기 추가: 1UP 아이템 획득 또는 특정 점수 달성
- 잔기 소진 시 게임 오버

### 사망 조건
- 어떤 크기의 풍선에 닿아도 **즉사**
- 제한 시간 초과 시 **즉사**
- 배리어 보유 중일 때는 1회 방어 후 배리어 소멸

### 2인 플레이 규칙
- 두 플레이어가 동시에 풍선에 닿으면 **Player 1만** 생명을 잃음
- 한 명이 사망하면 **두 플레이어 모두** 해당 스테이지 처음부터 재시작

---

## 풍선(버블)

### 크기와 분열

풍선은 4단계 크기로 존재하며, 하푼에 맞을 때마다 2개의 작은 풍선으로 분열된다.
가장 작은 크기(4단계)는 맞으면 완전히 제거된다.

```
[대]  →  [중]×2  →  [소]×4  →  [극소]×8  →  (제거)
```

### 이동 패턴
- 포물선을 그리며 바닥을 튀어 이동
- 크기가 클수록 높이 튀어오름
- 크기가 작을수록 빠르게 이동

---

## 하푼(무기)

### 기본 동작
- 발사 시 위쪽으로 직선 발사
- 천장 또는 블록에 닿으면 사라짐
- 풍선에 닿으면 풍선 분열 후 하푼 소멸

### 무기 종류
풍선 파괴 시 확률적으로 무기 아이템이 드롭된다. 모든 무기는 탄약 제한이 없다.

| 무기 | 설명 |
|------|------|
| 하푼 (기본) | 단발 발사, 천장 도달 시 소멸 |
| 더블 와이어 | 좌우 2발 동시 발사 |
| 파워 와이어 | 천장 또는 블록에 일정 시간 고정, 지속 판정 |
| 벌칸 미사일 | 짧은 간격으로 연속 속사 |

- 새 무기 획득 시 기존 무기를 대체
- 사망 시 기본 하푼으로 초기화

---

## 파워업 아이템

풍선 파괴 또는 스테이지 특정 시점에 드롭된다.

| 아이템 | 효과 | 비고 |
|--------|------|------|
| 배리어 | 풍선 충돌 1회 방어 | 방어 후 즉시 소멸 |
| 시계 | 모든 풍선 일시 정지 | 타이머는 계속 진행 |
| 모래시계 | 모든 풍선 이동 속도 감소 | 일정 시간 지속 |
| 다이나마이트 | 화면 내 모든 풍선을 즉시 최소 크기로 분열 | 오히려 풍선 수 급증 위험 |
| 1UP | 잔기 1개 추가 | |
| 음식 아이템 | 수천 점 보너스 점수 | 스테이지 특정 시점 드롭 |

---

## 스테이지 진행

### 클리어 조건
- 스테이지 내 **모든 풍선 제거**

### 실패 조건
- 잔기가 0인 상태에서 사망 → 게임 오버
- 게임 오버 시 메인 화면으로 복귀

### 스테이지 전환
- 클리어 후 다음 스테이지로 자동 전환
- 미션 내 마지막 스테이지 클리어 시 다음 미션으로 이동

---

## 점수

- 풍선 파괴 시 점수 획득 (크기가 클수록 고득점)
- 음식 아이템 획득 시 보너스 점수
- 현재 점수와 HI-SCORE를 화면 상단에 상시 표시
