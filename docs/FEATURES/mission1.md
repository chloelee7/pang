# Mission 1 — 후지산 (Mount Fuji)

## 개요

- **배경**: 일본 후지산
- **스테이지 수**: 3개 (Stage 1 → 2 → 3)
- **난이도**: 입문 (전체 게임 중 가장 쉬운 미션)

전체 게임의 첫 번째 미션으로, 게임 메커니즘을 익히기 위한 튜토리얼 성격을 가진다.

---

## 스테이지별 구성

### Stage 1
- **목표**: 팡 게임의 기본 조작 및 풍선 분열 메커니즘 학습
- **풍선 구성**: 대형 풍선 소수, 장애물 없음
- **제한 시간**: 60초
- **특징**: 평탄한 지형, 가장 단순한 구성

### Stage 2
- **목표**: 분열된 풍선을 동시에 처리하는 연습
- **풍선 구성**: 대형 + 중형 풍선 혼합
- **제한 시간**: 60초
- **특징**: Stage 1보다 풍선 수 증가

### Stage 3
- **목표**: Mission 1 최종 스테이지, 복수의 풍선 동시 처리
- **풍선 구성**: 다양한 크기의 풍선 다수
- **제한 시간**: 60초
- **특징**: Mission 1 중 가장 많은 풍선, 클리어 후 Mission 2로 진행

---

## 난이도 기준

| 항목 | Mission 1 설정 |
|------|---------------|
| 초기 풍선 수 | 적음 (2~4개) |
| 풍선 이동 속도 | 느림 |
| 제한 시간 | 60초 |
| 장애물 (발판) | 없음 |
| 파워업 드롭률 | 일반 |

---

## 진행 규칙

- Stage 1 클리어 → Stage 2 자동 전환
- Stage 2 클리어 → Stage 3 자동 전환
- Stage 3 클리어 → Mission 1 완료, Mission 2 (다음 지역) 진입
- 사망 시 현재 스테이지 처음부터 재시작 (잔기 1 감소)
- 잔기 소진 시 게임 오버 → 메인 화면 복귀

## 클리어 팁

- 대형 풍선은 코너로 몰리기 전에 먼저 터뜨려 분열 공간 확보
- 파워 와이어는 천장에 고정되어 지속 판정이 있으므로 다수 처리에 유리
- 다이나마이트는 모든 풍선이 최소 크기로 급증하므로 화면이 혼잡해질 수 있음
- 시계/모래시계 아이템은 우선 획득하여 안전하게 처리
