# 메인 화면 구성

## 개요

게임 진입 시 가장 먼저 표시되는 화면으로, 게임 시작 및 기본 설정 진입점 역할을 한다.

---

## 화면 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│           PANG                  │  ← 타이틀 로고
│                                 │
│        [ 1 PLAYER ]             │  ← 선택 중 (하이라이트)
│        [ 2 PLAYER ]             │
│                                 │
│      HI-SCORE  000000           │  ← 최고 점수
│                                 │
│   INSERT COIN / PRESS START     │  ← 안내 문구
│                                 │
└─────────────────────────────────┘
```

---

## 구성 요소

### 타이틀 로고
- 화면 상단 중앙에 "PANG" 텍스트 표시
- 게임 특유의 레트로 아케이드 스타일 폰트 적용

### 메뉴 항목
| 항목 | 동작 |
|------|------|
| 1 PLAYER | 1인 플레이 모드로 게임 시작 |
| 2 PLAYER | 2인 동시 플레이 모드로 게임 시작 |

- 방향키(↑↓) 또는 키보드로 항목 이동
- Enter 또는 스페이스로 선택 확정

### 최고 점수 (HI-SCORE)
- 현재 세션에서 달성한 최고 점수를 표시
- 최초 실행 시 `000000` 으로 초기화

### 안내 문구
- 게임 시작 방법을 하단에 안내
- 깜빡임 애니메이션 적용

---

## 전환 흐름

```
메인 화면
    │
    ├─ 1 PLAYER 선택 → Mission 1 Stage 1 시작
    └─ 2 PLAYER 선택 → Mission 1 Stage 1 시작 (2인)
```

---

## 키 입력

| 키 | 동작 |
|----|------|
| ↑ / ↓ | 메뉴 항목 이동 |
| Enter / Space | 선택 확정 |
