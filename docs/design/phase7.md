# Phase 7 설계 — HUD와 타이머

## 목표

점수·잔기·타이머를 화면에 표시하고, 60초가 다 되면 즉사 처리.

## 점수 체계

| 레벨 | 이름 | 점수 |
|------|------|------|
| 3 | 대 | 100 |
| 2 | 중 | 200 |
| 1 | 소 | 400 |
| 0 | 극소 | 800 |

분열된 자식 풍선을 터뜨릴 때 각각 점수 획득.

## 타이머

- `stageTimer = 60` 초
- 매 프레임 dt만큼 감소
- 0 이하 → loseLife() 호출
- 스테이지 재시작 or 클리어 시 60초 리셋

## HUD 레이아웃

```
SCORE  012345        TIME  60
♥ ♥ ♥
```

상단 좌측: SCORE, 상단 우측: TIME, 잔기 아이콘(♥) 좌측 두 번째 줄
