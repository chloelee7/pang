# PRD: 팡(PANG) 게임 - Mission 1

## 개요

**팡(Pang)** 은 1989년 Mitchell Corporation이 개발한 아케이드 게임이다.
플레이어는 하푼(작살)을 발사해 포물선을 그리며 바닥을 튀는 풍선(버블)을 모두 터뜨려야 한다.

**Mission 1** 은 일본 후지산을 배경으로 한 스테이지 1~3으로 구성된 첫 번째 미션이다.
전체 게임은 17개 지역, 총 50개 스테이지로 이루어진다.

---

## 핵심 게임플레이

### 목표
스테이지 내 모든 풍선을 제한 시간 안에 제거한다.

### 플레이어 조작
- 좌우 이동
- 하푼 발사 (위 방향, 버튼 1개)
- 1인 또는 2인 동시 플레이 지원

### 승패 조건
| 조건 | 결과 |
|------|------|
| 모든 풍선 제거 | 스테이지 클리어 |
| 풍선에 닿음 | 즉사 (배리어 보유 시 1회 방어) |
| 제한 시간 초과 | 스테이지 실패 |

---

## 풍선(버블) 메커니즘

- 풍선은 **4가지 크기(대 → 소)**로 존재
- 하푼에 맞으면 **2개의 더 작은 풍선으로 분열**
- 가장 작은 크기의 풍선은 맞으면 **완전히 제거**
- 모든 풍선은 포물선을 그리며 바닥을 튀어 이동

```
대형 → 중형×2 → 소형×4 → 극소×8 → 제거
```

---

## 무기 시스템

플레이어는 기본 하푼으로 시작하며, 풍선을 터뜨릴 때 무기 아이템이 드롭될 수 있다.
모든 무기는 탄약 제한이 없다.

| 무기 | 설명 |
|------|------|
| 하푼 (기본) | 단발 발사, 천장에 닿으면 사라짐 |
| 더블 와이어 | 2발 동시 발사 |
| 파워 와이어 | 천장 또는 블록에 일정 시간 고정 |
| 벌칸 미사일 | 연속 속사 |

---

## 파워업 아이템

풍선을 터뜨리거나 스테이지 특정 시점에 아이템이 드롭된다.

| 아이템 | 효과 |
|--------|------|
| 배리어 | 풍선 충돌 1회 방어 후 소멸 |
| 시계 | 풍선 일시 정지 |
| 모래시계 | 풍선 이동 속도 감소 |
| 다이나마이트 | 화면 내 모든 풍선을 즉시 최소 크기로 분열 (위험) |
| 1UP | 잔기 1개 추가 |
| 음식 아이템 | 수천 점 보너스 점수 |

---

## 생명 시스템

- 초기 잔기: 3개
- 1UP 아이템 또는 점수 달성으로 추가 잔기 획득
- 2인 플레이 시 한 명이 사망하면 **두 플레이어 모두 해당 스테이지를 처음부터 재시작**
- 2인 플레이 시 동시에 풍선에 닿으면 **Player 1만 생명을 잃음**

## Mission 1 스테이지 구성

| 스테이지 | 배경 | 특징 |
|----------|------|------|
| Stage 1 | 후지산 | 튜토리얼 성격, 가장 적은 풍선 |
| Stage 2 | 후지산 | 풍선 수 증가 |
| Stage 3 | 후지산 | Mission 1 최종 스테이지 |
